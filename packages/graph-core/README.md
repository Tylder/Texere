# graph-core

Canonical graph types, IDs, and policy schemas.

## Scope

- Package: graph-core
- Runtime: shared (node/esm)
- Scope tag: scope:graph

## Responsibilities

- Define node/edge shapes with schema_version\n- Provide deterministic ID helpers\n- Provide policy
  kinds and selection inputs

## Non-goals

- Storage adapters or persistence\n- Ingestion orchestration\n- Projection execution
