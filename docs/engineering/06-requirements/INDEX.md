## How to add a requirement

1. Create a new file: `REQ-<domain>-<id>.md`
2. Include:
   - MUST/SHOULD/MAY statement
   - Measurable fit criteria
   - Verification method
   - Traceability link(s) to PROBDEF/DRFC/XRFC/RFC
3. Update this INDEX.md with metadata
4. Link the requirement from its traceability source (DRFC, XRFC, RFC, Spec, etc.)

See `/docs/engineering/_templates/REQ-template.md` for a template.
