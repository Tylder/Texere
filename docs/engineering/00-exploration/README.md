# 00-Exploration

**Purpose:** Capture raw discovery and brainstorming sessions.

Exploration logs are intentionally **disposable**. They capture messy thinking, alternative
framings, half-formed ideas, and unresolved contradictions before they converge into durable
artifacts.

## File naming

```
EXPLOG-YYYY-MM-DD-<topic>.md
```

Examples:

- `EXPLOG-2025-01-20-workflow-resumability.md`
- `EXPLOG-2025-01-19-error-handling-strategy.md`

## Rules

- Never treated as canonical truth
- Must end with a **Synthesis Delta** section that captures:
  - New facts discovered
  - New assumptions introduced
  - Unknowns created or closed
  - Candidate problems to add/update in PROBDEF
  - Candidate journeys to add/update in XRFC
- May be archived or discarded freely
