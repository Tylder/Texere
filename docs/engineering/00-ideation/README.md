## How to contribute

For each feature or initiative, create three documents:

1. `IDEATION-PROBLEMS-<feature>.md` – Problems, failure modes, scenarios
2. `IDEATION-EXPERIENCE-<feature>.md` – Personas, journeys, invariants
3. `IDEATION-UNKNOWNS-<feature>.md` – Open questions, assumptions, blockers

See `../_templates/IDEATION-*-template.md` for templates.

- [ ] Update `../DOCUMENT-REGISTRY.md` (add entry)
