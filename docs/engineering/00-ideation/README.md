Ideation documents are **disposable** by design—they should be archived (not deleted) once
Requirements are finalized to preserve history and prevent link rot.

## How to contribute

For each feature or initiative, create three documents:

1. `IDEATION-<feature>-problems.md` – Problems, failure modes, scenarios
2. `IDEATION-<feature>-experience.md` – Personas, journeys, invariants
3. `IDEATION-<feature>-unknowns.md` – Open questions, assumptions, blockers

See `../_templates/IDEATION-template-*.md` for templates.

After creating a document:

- [ ] Update this README (add to Active section)
- [ ] Update `../DOCUMENT-REGISTRY.md` (add entry)
